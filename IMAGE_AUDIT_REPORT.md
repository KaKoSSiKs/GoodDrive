## Рекомендации по замене

### Для логотипа (logo.jpg)

**Вариант 1: Создать собственный логотип**
- Использовать онлайн-генераторы: Canva, LogoMaker, Hatchful
- Создать в графическом редакторе (Figma, Adobe Illustrator)
- Заказать у дизайнера

**Вариант 2: Использовать только текст**
- Красивый шрифт + цветовая схема сайта
- Можно добавить простую геометрическую форму

### Для фонового изображения (img_car.jpg)

**Бесплатные источники изображений:**
- [Unsplash](https://unsplash.com) - ключевые слова: "auto parts", "car repair", "automotive"
- [Pexels](https://pexels.com) - ключевые слова: "car parts", "mechanic", "garage"
- [Pixabay](https://pixabay.com) - "automobile", "spare parts"

**Альтернатива:**
- Использовать градиентный фон вместо изображения
- Создать абстрактный паттерн с автомобильной тематикой

### Для иконок категорий

**Бесплатные источники иконок (без водяных знаков):**

1. **[Heroicons](https://heroicons.com/)** - MIT License
   - Minimalist SVG icons
   - Идеально интегрируется с Tailwind CSS

2. **[Lucide Icons](https://lucide.dev/)** - ISC License
   - Уже используется в проекте! (см. package.json)
   - Более 1000 иконок

3. **[Iconoir](https://iconoir.com/)** - MIT License
   - Чистые, современные иконки

4. **[Tabler Icons](https://tabler-icons.io/)** - MIT License
   - 4000+ иконок SVG

5. **[Font Awesome Free](https://fontawesome.com/search?o=r&m=free)** - Free License
   - Классический набор иконок

6. **[Feather Icons](https://feathericons.com/)** - MIT License
   - Простые и элегантные

## Как заменить иконки на Lucide (уже в проекте!)

Вместо PNG файлов можно использовать компоненты Lucide:

```svelte
<script>
  import { 
    CircuitBoard,  // Электроника
    Engine,        // Двигатель  
    ShieldCheck,   // Тормоза
    Wrench,        // Подвеска
    ShoppingCart   // Корзина
  } from 'lucide-svelte';
</script>

<!-- Пример использования -->
<CircuitBoard size={40} color="#8B0000" />
```

## Действия по замене

### Шаг 1: Проверьте открытые изображения
Все изображения были открыты в вашем просмотрщике. Проверьте каждое на наличие:
- Водяных знаков (watermark)
- Текста с названиями сайтов (Shutterstock, Getty Images, iStock, etc.)
- Логотипов сторонних компаний
- Надписей "SAMPLE", "PREVIEW", "DEMO"

### Шаг 2: Замените проблемные изображения

**Если на изображениях есть водяные знаки:**

1. Для иконок - используйте Lucide Icons (уже в проекте)
2. Для логотипа - создайте новый или используйте текстовый
3. Для фона - скачайте бесплатное изображение или используйте градиент
